# CineVerseX

CineVerseX is a movie ticket booking web application built with Flask, MySQL, HTML, CSS, Bootstrap, and JavaScript.

## Features

- User registration and login
- Admin and user roles
- Movie management
- Theater and screen management
- Show management
- Seat selection
- Ticket booking
- QR ticket generation
- PDF ticket download
- Admin dashboard
- Revenue tracking
- Movie search

## Tech Stack

- Python Flask
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLAlchemy

## How to Run

```bash
cd backend
python app.py
